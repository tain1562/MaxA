# React Native Project

This is a React Native project.

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- React Native CLI
- Android Studio (for Android development)
- Xcode (for iOS development, macOS only)

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   # or
   yarn install
   ```
3. Start the development server:
   ```bash
   npm start
   # or
   yarn start
   ```

### Running the App

- For Android:
  ```bash
  npm run android
  # or
  yarn android
  ```
- For iOS:
  ```bash
  npm run ios
  # or
  yarn ios
  ```
